# CookiStocker-mod-script-clone
Github-based upload of the CookiStocker mod for Cookie Clicker


A rehost of the CookiStocker mod for Cookie Clicker. The original is seemingly only available as a pastebin link and will not load on the web version of the game.

I am currently attempting to see if putting it in this repo format like other web-based mods will let me use the mod properly on the web version.

The script is not my creation and all credit for it goes to GingerGuy and z on Steam. See all about it here: https://steamcommunity.com/sharedfiles/filedetails/?id=2599187047
If they happen upon this project and would prefer it removed, please let me know and I will happily oblige.
